ULTROvpn — Bot + Webhook (V2Ray vmess + HTTP proxy generator)

Setup:
1) Place logo.png into project root (optional).
2) Create .env with BOT_TOKEN, ADMIN_IDS, PAY_WEBHOOK_SECRET, etc. or edit config.py.
3) Install dependencies:
   pip install -r requirements.txt
4) Run:
   - With Docker: docker compose up -d --build
